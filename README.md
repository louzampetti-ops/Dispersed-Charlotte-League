# DCL Fantasy Football Site

Website for The Dispersed Charlotte League.

## Pages
- Home
- Current Season
- Season History
- All-Time
- Teams
- League Directory

## Main folders
- assets/ = logo and images
- css/ = styling
- js/ = site data and rendering logic
- seasons/ = archived season pages
- franchises/ = one page per franchise

## How to update
1. Edit `js/data.js`
2. Commit changes in GitHub
3. Refresh the live site after GitHub Pages republishes
